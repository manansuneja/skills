## How to add a meeting

Fast path: paste, drag, or save notes anywhere under `{{DOCS_ROOT}}/meetings/`, then tell the Chief
the Workspace Chief to process the notes. It will create the dated meeting folder, preserve the raw notes as
`raw/raw-notes.md`, remove the loose inbox copy, write `summary.md`, and add a row here.

Manual shape, if you prefer:

1. Create `{{DOCS_ROOT}}/meetings/<MMM-DD-YYYY>-<short-title>/`.
2. Put your notes in `raw/raw-notes.md` (type or paste - messy is fine).
3. Tell the Workspace Chief: *"Process these meeting notes."* It writes `summary.md` and adds a row here.
