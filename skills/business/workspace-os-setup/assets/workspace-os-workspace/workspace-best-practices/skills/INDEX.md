# workspace-best-practices/skills/ — Index

> **For you and your agents:** Skills strongly shape output quality, structure, and format. Tell the
> Workspace Chief what you want changed; the agent handles the files and routing.

Start small. These common skills cover the generic core:

| Skill | Use it to | Applied by |
|---|---|---|
| [summarize-notes.md](summarize-notes.md) | Turn raw notes into a clean, structured summary | meeting-summarizer |
| [brainstorm.md](brainstorm.md) | Explore options and converge with trade-offs | brainstorm-partner |
| [synthesize-outcomes.md](synthesize-outcomes.md) | Create durable insights, recommendations, and priorities | outcome-synthesizer |
| [document-context.md](document-context.md) | Keep durable context, profile, and indexes current | documentation-steward |

Customization should add, adapt, merge, or remove skills with the content structure. Category-specific
skills—for proposals, sourcing, content production, study, delivery, metrics, or another recurring
workflow—belong here only when this workspace needs them.

Any skill change must update its templates/references, matching specialist when useful, Workspace
Chief routing, and affected indexes. Follow
[manage-workspace-skills.md](../../_workspace_setup_docs/skills/manage-workspace-skills.md).
