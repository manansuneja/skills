# workspace-best-practices/ — Index

> **For you and your agents:** This is the workspace customization center. Describe your intention;
> the agent updates and connects the files for you.

| Folder | What it controls |
|---|---|
| [skills/](skills/INDEX.md) | Repeatable judgment, workflow, output structure, and formatting rules |
| [templates/](templates/INDEX.md) | Exact reusable output shapes |
| [references/](references/INDEX.md) | Examples, sources, voice, and style samples |

You can open this folder and ask naturally:

- “Make meeting summaries shorter and put decisions first.”
- “Create a skill and template for interior-design project briefs.”
- “Use this proposal as the format and tone for future proposals.”
- “Remove any skill or template this workspace no longer needs.”

No configuration knowledge is required. The agent maintains indexes, links, specialists, and routing.
