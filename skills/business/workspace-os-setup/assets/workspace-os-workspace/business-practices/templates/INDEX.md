# templates/ - Index

Templates are reusable blank structures that skills should follow when creating artifacts. They can
be Markdown, JSON, or another structured format if that is what the owner wants.

When a matching template exists here, use it before the default structure inside an individual skill.
If a template changes, update this index and any skill that should prefer it.

| Template | Use it for | File |
|---|---|---|
| Meeting summary | Filed meeting summaries | [meeting-summary.md](meeting-summary.md) |
| Outcome | Recommendations, prioritization, scope cuts, and synthesis | [outcome.md](outcome.md) |
| Offer brief overview | Area, package, initiative, or project-set overview | [project-brief.md](project-brief.md) |
| Offer brief | Offer, package, or deliverable brief | [offer-brief.md](offer-brief.md) |
| Client story | Client story, acceptance criteria, and priority | [client-story.md](client-story.md) |
| Decision | Decision record and rationale | [decision.md](decision.md) |
| Brand and delivery note | UX, brand, delivery considerations, and trade-offs | [brand-delivery-note.md](brand-delivery-note.md) |
| Metric insight | Metric, experiment, or evidence readout | [metric-insight.md](metric-insight.md) |
| Plan | Plan, release plan, or sequencing note | [plan.md](plan.md) |
| Client communication | Exec update, client summary, launch note, or brief | [client-communication.md](client-communication.md) |
